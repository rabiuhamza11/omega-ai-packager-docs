OMEGA AI Packager

Bundle OMEGA INFINITY AI agent projects into deployable packages, and ship them straight to GitHub, Vercel, Render, Netlify, and Railway.

Install

npm install
npm run build
npm link

1. Scaffold a manifest

Creates an omega.agent.yml describing your agent — its role, entry point, dependencies, and deploy targets.

omega-pack init my-agent
name: my-agent
version: 0.1.0
role: Backend
entryPoint: src/index.ts
dependencies: []
env: {}
targets:
  - github
  - vercel

2. Package it

Produces a .zip archive plus a JSON manifest, ready to hand to any deployment pipeline or Docker host.

omega-pack build --docker --env-template

3. Deploy it

Sends the packaged files straight to the DeployForge multi-platform deployment function — pushing to GitHub, creating a Vercel project, spinning up a Render static site, deploying to Netlify, and provisioning a Railway project, all in one call.

omega-pack deploy
